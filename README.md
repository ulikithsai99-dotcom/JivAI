# JivAI

From Panic to Action

JivAI is an AI-powered crisis response companion that helps citizens navigate emergencies through voice-first interaction, government knowledge integration, and real-time action guidance.

## Features

- Voice-first emergency reporting
- AI emergency classification
- Government knowledge engine
- Multilingual support
- Offline emergency guidance
- Accessibility-first design
- Emergency escalation support

## Tech Stack

Frontend:
- React
- TypeScript

Backend:
- FastAPI
- Python

AI:
- Gemini API
- RAG Architecture
- FAISS

## Mission

Reduce decision fatigue during emergencies and help citizens receive assistance faster.
